# APP Enhancement Proposal (AEP) Template

Use this template to propose changes or extensions to the Applicant Profile Protocol.

## AEP Number
AEP-XXXX (assigned upon acceptance)

## Title
Short, descriptive title of the proposal.

## Author(s)
Name, contact (GitHub handle, email)

## Status
Draft | Review | Accepted | Rejected | Superseded

## Summary
One-paragraph summary of the problem and the proposed solution.

## Motivation
- What problem does this solve?
- Who benefits?
- Why now?

## Specification
- Detailed semantics and JSON structure
- JSON Schema diff (if applicable)
- Validation rules
- Backward-compatibility considerations

## Rationale
- Alternatives considered
- Trade-offs
- Prior art (links to other standards/schemas)

## Security & Privacy
- PII impact
- Mitigations

## Implementation
- Suggested reference implementation
- Exporter impacts (JSON Resume / Europass / HR-XML)

## Migration
- Steps for consumers to adopt the change
- Deprecation notes

## Appendix
- Examples
- Diagrams
