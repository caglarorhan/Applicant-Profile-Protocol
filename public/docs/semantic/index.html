Semantic Layer

Linked data and semantic web integration for APP

The semantic layer extends APP with JSON-LD support, enabling integration with the semantic web and linked data ecosystems.

By adding a @context to your APP profile, you can:

  • Link to standardized vocabularies (Schema.org, FOAF, etc.)
  • Enable machine-readable semantic relationships
  • Integrate with knowledge graphs and RDF systems
  • Improve discoverability and interoperability
